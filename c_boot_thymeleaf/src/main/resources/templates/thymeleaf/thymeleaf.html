1. 변수를 선언하고 출력해보기

	변수선언 => th:with
	변수출력 => ${...}
	선언된 변수의 스코프는 해당 변수가 선언된 html 태그이다. 형제태그나 부모태그에서는 사용할 수 없다.
	변수를 문자열 리터럴과 함께 사용할 때는 |abc${...}|형태로 사용한다.
